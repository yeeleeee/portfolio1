Open Learning Initiative
Open Learning Initiative is an open source web application that lets faculties create, share and analyze their courses. As a UX designer, I conduct user research, develop UX/UI design for educational authoring tool.
This is an ongoing project. Learn more
About the project
Duration: ongoing
Team size: 10~
Reaches: as many as it can..!
I was the only designer in the development team, which gave me an opportunity to get to know the developers' work process (and jokes).

    What I learned from working with developers:

  • What factors go into a prioritization of tasks: user feedback, a calculated effort of the team, urgency.
  • Thinking about the edge cases.
  • Agile development and reporting out on daily basis.
  • Standards/style guides we have to think about when actually making products.
  • Joining the old style and the new one is really difficult.
A piece of my presentation material.
My responsibility included user interviews, user research and presenting the insight and solutions to the team, UI design and pitches.

    What I learned from developing an authoring tool:

  • Thinking about the edge cases, not the best possible way as it can happen.
  • Handoff to developers - they accepted png files but they would need more than that for a new platform.
  • Making design compromises are sometimes necessary to ensure the users feel heard..
